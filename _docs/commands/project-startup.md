## **Setup Instructions**

### **Linux**

1. Crate a virtual environment:

```bash
python venv venv
source venv/bin/active
```

in case you (me) are using `fish`:

```bash
source venv/bin/activate.fish
```

2. Install FastAPI and Uvicorn:

```bash
pip install fastapi uvicorn
pip install "uvicorn[standard]"
```
